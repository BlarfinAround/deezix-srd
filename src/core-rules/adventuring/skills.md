# Skills

Skills use the [X-in-6 mechanic](../character/the-x-in-6-mechanic.md): roll 1d6 and succeed if the result is equal to or less than the skill's target number.

## General Skills

Every character begins with four **General Skills**, each at a **1-in-6 chance**.

### Break
Forcing open stuck or barred doors, smashing locked chests, bending bars.

- Roll when a door or container resists normal opening attempts.
- Heavily reinforced objects may impose −1 or −2.

### Languages
Speaking, reading, or understanding a language outside the character's known tongues.

- Roll when communicating in an unfamiliar language or deciphering an old text.
- Success grants a rough understanding of the language which may be added to the character sheet.

### Search/Listen {#search-listen}
Finding hidden things and detecting sounds: secret doors, concealed traps, approaching monsters, overheard conversations.

- Roll when actively examining an area (takes 1 turn) or listening at a surface or distance.

### Tumble
Negating fall damage and disengaging from melee.

- **Fall**: On a successful roll, negate 1d6 of fall damage.
- **Disengage**: Safely escape a melee engagement at full speed. See [Disengage](../adventuring/combat.md#disengage).

## Class Skill Bonuses

Each class applies its key ability score modifier to a designated General Skill (minimum 1, maximum 5).

| Class      | Key Ability → General Skill |
| ---------- | --------------------------- |
| Fighter    | STR → Break                 |
| Cleric     | WIS → Search/Listen         |
| Magic-User | INT → Languages             |
| Thief      | DEX → Tumble                |

The Cleric, Magic-User, and Thief also have **class skills** described in their entries:

- [Cleric](../character/classes/cleric.md): Turn Undead
- [Magic-User](../character/classes/magic-user.md): Arcane Lore
- [Thief](../character/classes/thief.md): Climb, Tinker, Stealth, Sleight of Hand (plus skill points to allocate)

::: {.callout-important title="Skill Maximum"}
No skill may exceed a **5-in-6 chance**. A **1 always succeeds** and a **6 always fails**.
:::

## Using Skills

The referee calls for a skill roll when the outcome is uncertain and failure is interesting. Most dungeon skill rolls take one turn (10 min).
