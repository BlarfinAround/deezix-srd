# Saving Throws

When a character is exposed to a dangerous magical or physical effect, they roll a **saving throw** to resist or reduce it. Saving throws use the [X-in-6 mechanic](../character/the-x-in-6-mechanic.md).

::: {.callout-important title="Saving Throw"}
Roll 1d6. If the result is **equal to or less than** the character's save value for that category, the throw succeeds. A **1 always succeeds** and a **6 always fails**.
:::


## The Five Categories

| Category      | Covers                                       |
| ------------- | -------------------------------------------- |
| **Death**     | Death effects, poison, instant kills         |
| **Wands**     | Wand and ray attacks, targeted magical beams |
| **Paralysis** | Hold, paralysis, petrification               |
| **Breath**    | Breath weapons, area explosions, blasts      |
| **Spells**    | Spells, staves, rods, magical devices        |

A spell entry specifies which category to roll against. When unclear, the referee picks the closest match.

## Saving Throw Values

**Character** save values are listed in their respective class description.

**Monster** save values are usually given in their stat block and normally equal a fighter of a level matching their hit dice.
