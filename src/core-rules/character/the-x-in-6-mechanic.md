# The X-in-6 Mechanic

Deezix makes extensive use of X-in-6 rolls:

::: {.callout-important title="X-in-6 Roll"}
Roll 1d6. Succeed if the result is **≤ X**.

A **6 always fails** and a **1 always succeeds**, regardless of any modifiers.
:::

## Reading the Odds

The practical range for any roll is a **1-in-6 chance** to a **5-in-6 chance**. When success or failure is guaranteed, no roll is required.

| Target | % Chance | Likelihood |
| ------ | :------- | ---------- |
| 1-in-6 | 16.7%    | Highly unlikely |
| 2-in-6 | 33.3%    | Probably not |
| 3-in-6 | 50.0%    | Maybe |
| 4-in-6 | 66.7%    | Probably |
| 5-in-6 | 83.3%    | Most likely |

## Modifiers

Ability scores, class features, and situational factors modify the target number by **±1** (sometimes ±2 for extreme circumstances).

::: {.callout-note title="Example: Stealth in Torchlight"}
Rowan the Thief has a Stealth skill of 3-in-6. In bright torchlight, the referee judges he is at a disadvantage and applies −1, reducing his effective target to a 2-in-6 chance. He rolls a 2 — success.
:::

Modifiers **never reduce the target below 1** and **never increase it above 5**.
