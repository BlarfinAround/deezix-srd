# Experience and Advancement

Characters grow in power by accumulating **Experience Points (XP)**. XP is earned primarily by recovering treasure and returning it to safety.

## Earning XP

### Gold for XP
The primary source of XP is **treasure**:

::: {.callout-important title="Gold for XP"}
Each **gold piece (gp) of value returned to civilization = 1 XP**.
:::

Gold must be returned to a safe haven to count.

The gp value of recovered treasure:

- **Coins**: as listed in the [Coin Values](../referee/treasure.md#coin-values) table.
- **Gems, jewelry, art**: appraised gp value.
- **Mundane equipment**: half listed value (when sold).

### Monster XP
Monsters grant a small amount of XP for being defeated (slain, driven off, or significantly overcome):

| Monster HD  | Base XP | Bonus per Ability |
|-------------|---------|-------------------|
| Less than 1 |     5   |       1           |
| 1           |    10   |       3           |
| 1+          |    15   |       4           |
| 2           |    20   |       5           |
| 2+          |    25   |      10           |
| 3           |    35   |      15           |
| 3+          |    50   |      25           |
| 4           |    75   |      50           |
| 4+          |   125   |      75           |
| 5           |   175   |     125           |
| 5+          |   225   |     175           |
| 6           |   275   |     225           |
| 6+          |   350   |     300           |
| 7           |   450   |     400           |
| 8           |   650   |     550           |
| 9–10        |   900   |     700           |
| 11–12       | 1,100   |     800           |
| 13–16       | 1,350   |     950           |
| 17–20       | 2,100   |   1,150           |
| 21+         | 2,400   |   1,350           |

Each asterisk on a monster's HD (e.g., HD 2\*\*) marks a special ability that meaningfully raises its threat — not every trait is counted. Add the row's **Bonus per Ability** to the **Base XP** once per asterisk.

## Dividing XP

Split equally among all PCs and retainers who participated and survived. Retainers receive **half a share**.

::: {.callout-note title="Example: Dividing XP"}
3 PCs + 1 retainer, 925 XP total: 3.5 shares → 264 XP each PC, 132 XP to retainer.
:::

## Advancement Tables

Full per-level tables (XP required, HD, Attack Bonus, saves, and class features) appear in each class entry.

## Leveling Up

Leveling up occurs between sessions in a safe haven. On gaining a level: roll new Hit Die + CON mod (min 1), or add fixed hit points at level 10+; record new AB, saves, and any new class features.
