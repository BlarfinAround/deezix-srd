# Encumbrance

Characters can carry up to **10 items + STR modifier** without penalty. Count carried items to determine movement.

| Condition  | Items Carried          | Effect                     |
| ---------- | ---------------------- | -------------------------- |
| Normal     | 1 to (10 + STR mod)    | Full movement              |
| Encumbered | up to (15 + STR mod)   | Halved movement            |
| Overloaded | (16 + STR mod) or more | Cannot move or cast spells |

## What Counts as an Item

Characters are assumed to have whatever mundane means of carrying gear is appropriate to the setting — backpacks, sacks, belt pouches, bandoliers — at no cost and with no slot.

**Standard (1 item)**: Most equipment — a weapon, lantern, rope, tool, waterskin, spellbook.

**Bundled (1 item per bundle)**: Small or stackable items packed together. Examples:

| Bundle  | Limit                                        |
| ------- | -------------------------------------------- |
| Arrows  | ×20                                          |
| Bolts   | ×20                                          |
| Torches | ×6                                           |
| Rations | ×7 days                                      |
| Coins   | up to 300; each additional 300 = 1 more item |

**Oversized (3 items)**: Large or unwieldy objects: 10' pole, armor carried in a pack.

**Worn armor**: Counts as **0 items**. A shield always counts as 1 item.

### Notation

In equipment tables, *italic item names* mark bundleable items and ***bold-italic names*** mark oversized items.

## Pack Animals and Carts

| Transport        | Item Capacity |
| ---------------- | ------------- |
| Pack mule        | 30 items      |
| Draft horse      | 40 items      |
| Cart (1 horse)   | 80 items      |
| Wagon (2 horses) | 200 items     |
